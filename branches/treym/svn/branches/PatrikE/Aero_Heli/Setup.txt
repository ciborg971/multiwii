
/******************************************************/
/****Importent notes...********************************/
/******************************************************/

The motor must be ARMED in same way as the multis.
Pasthrumod can be used on all three models.

For safetyreasons...
If Engine is used it MUST be connected to servo[7].

/******************************************************/



/******************************************************/
/**** Setup *******************************************/
/******************************************************/

In config:
// For heli
#define CollectivePitch AUX2 	// Or what channel you use as collective.


/* Same function to adjust tail on Helicopter */ 
/* you can change the tricopter servo travel here */
#define TRI_YAW_CONSTRAINT_MIN 1020
#define TRI_YAW_CONSTRAINT_MAX 2000
#define TRI_YAW_MIDDLE 1500 // tail servo center pos. - use this for initial trim; later trim midpoint via LCD

/******************************************************/
	Setup Airplane
/******************************************************/

Connect the servos.

Setup your TX.
Set servo directions in PassthruMode.
Expo should be set on TX. 
Set EXPO to 0 in gui.

Setup MWii in MultiWii-tab
(  Servosettings For Airplane  )
Gyro/Acc mode.
Set servoreverse for Gyro if needed.
Adjust ServoMid & Servotravel.


/******************************************************/


In Output:
Adjust the mixes to suite your servosetup.

/******************************************************/
